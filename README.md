# 📚 نظام مراقبة حلقات القرآن

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/your-username/quran-groups-tracker)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

نظام متكامل لتتبع تغييرات أعضاء مجموعات واتساب الخاصة بحلقات تحفيظ القرآن الكريم.

## ✨ المميزات

- ✅ تتبع **المغادرين** من المجموعات تلقائياً
- ✅ تتبع **المنضمين** الجدد
- ✅ إشعارات فورية عبر الواجهة
- ✅ عرض عدد الأعضاء لكل حلقة
- ✅ تحديث تلقائي كل 30 ثانية
- ✅ حفظ سجل التغييرات
- ✅ واجهة عربية بالكامل
- ✅ تصميم متجاوب مع جميع الأجهزة

## 🛠️ التقنيات المستخدمة

- **Node.js** - بيئة التشغيل
- **whatsapp-web.js** - للاتصال بواتساب
- **Express.js** - خادم الويب
- **Socket.IO** - الاتصال الفوري
- **Puppeteer** - التحكم بالمتصفح

## 📋 المتطلبات

- Node.js 14+ 
- متصفح حديث
- حساب واتساب نشط
